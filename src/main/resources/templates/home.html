Welcome...

This project has been started to provide the community with a tool to find Landing Sites for missions on Surviving Mars. At the moment, this is a work-in progress, but please feel free to suggest features/improvements on GitHub. However, this is a hobby project and other things come first.

The majority of pages link back to here for now... but as features get added I will try to keep the change notes updated in the about page, as well as on github.
